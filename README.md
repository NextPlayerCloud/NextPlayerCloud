# OpenList + MPV 播放器组合方案

这是一个将 **OpenList** 与 **MPV 播放器**完美结合的本地播放解决方案，支持挂载各种云盘（阿里云盘、OneDrive、Google Drive、WebDAV、SMB 等），实现高速、流畅、无广告的本地化播放体验。

## 特性亮点
- OpenList Web可视化操作
- 支持几乎所有主流网盘（数百种驱动）
- 通过 WebDAV / SMB 协议挂载，MPV 可直接播放网盘文件
- 完全本地播放，无需转码、无广告、支持硬解
- 原生支持字幕自动加载、外挂字幕、播放记录等
- 播放流视频自动缓冲左滑滑动如播放本地视频般丝滑
- 支持自动下一集
